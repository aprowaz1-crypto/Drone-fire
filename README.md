# Drone-fire

Цей репозиторій містить базову Android Studio аплікацію для моніторингу аудіосигналів.

Інструкція:

1. Відкрий папку `Drone-fire` в Android Studio.
2. Імпортуй проєкт як Gradle проект.
3. Переконайся, що мінімальна версія Android встановлена на API 29.
4. Запусти додаток на пристрої або емульяторі.

Проєкт включає:

- `AndroidManifest.xml` з необхідними дозволами
- фонову службу для аудіомоніторингу
- обробку спектру 40-100 Гц
- повноекранні тривоги та логування
- GitHub Actions workflow для автоматичної збірки APK
